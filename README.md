# ds-tutorial-tanks

## Install

```
Start the server

1. cd to the project
2. cd ds-server
3. npm install deepstream.io
3. node start.js

Start the browser game in another terminal window

1. cd to the project
2. cd browser
3. npm install -g bower
3. bower install
4. python3 -m http.server
5. connect to localhost:8000?tankName=<YourName>

```
